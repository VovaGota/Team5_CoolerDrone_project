# Team5_CoolerDrone_project
Название команды: Ядерный холодильник Состав команды: Крапля Владимир Михайлович, Грошев Михаил Иванович, Пронюшкин Илья Александрович. Задача №1  Разработать беспилотный летательный аппарат на базе предоставленной электроники для реализации «холодовой цепи» доставки медикаментов в отдалённые и горные поселения

**Принцип работы устройства.**
Наше решение представляет из себя прикрепляемую к fpv дрону систему сброса контейнеров с медикаментами
На дрон ставится снизу в раму крепление типа «Ласточкин хвост», к данному креплению присоединяют основную систему и закрепляют фиксирующим болтом с гайкой. Систему охлаждения подключают к энергетике дрона, через вход платы распределения питания.  Далее на лучи устанавливаются специальные ножки, а на систему сброса резиновой лентой присоединяются 2 контейнера с медикаментами. При запуске дрона начинает работать система охлаждения. Команда на сброс подаётся с полетного контроллера, с него же через Arduino Nano регулируется температура внутри контейнера.

Контейнеры представляют из себя обитые поролоном в канавках снаружи и сплошным слоем внутри пластмассовые эллипсоидные контейнеры для с отверстием для введения термодатчика и кулера охлаждения. Один контейнер предназначен под инфузионный пакет, второй под ампулы и блистеры с таблетками. Также предусмотрены внутренние вставки под удобное хранение ампул.

При запуске дрона подключенная к его энергетике система охлаждения, в виде элемента Пельтье и кулера запускается через модуль понижения напряжения. С элемента Пельтье выводится тепло через крупный верхний радиатор, для ускорения сбора тепла также на термопасту посажен дополнительный малый радиатор снизу. Холодный воздух отправляется через сопло в контейнер с помощью кулера. Воздух попадает в систему через воздухозаборники между элементом Пельтье  

**Содержание репозитория**
В коде представлен контроль температуры в пределах от 2 до 15 градусов по Цельсию, также контроль системы охлаждения в виде подачи определенного тока на элемент Пельтье. И контроль сервоприводов сброса контейнеров.

# Система контроля температуры
- Контроллер: SKL-F405MRH-S
- Датчик: DHT11
- Нагрузка: Элемент Пельтье 12V 55W
- Диапазон: +2°C до +15°C
